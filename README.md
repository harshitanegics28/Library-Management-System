# Library Management System

## Project Description
This project is a Library Management System developed using DBMS concepts. It manages books, members, issue/return records, and fines efficiently.

## Features
- Book Management
- Member Management
- Issue & Return Tracking
- Fine Calculation

## Technologies Used
- MySQL Workbench
- SQL

## Files Included
- Project_Report.pdf
- Presentation.pptx
- library_database.sql

## How to Run
1. Open MySQL Workbench
2. Create a new database
3. Import the SQL file
4. Execute the script
